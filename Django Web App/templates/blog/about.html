About Page

ESPlantMonitoring aims to provide an easy way of monitoring multiple plants using standalone ESP devices. Data is sent via MQTT and stored. Realtime data can be accessed via an MQTT subscriber or near realtime data can be view using the web interface. In order to use the service, you have to sign up. You can then use the registered credentials to access the MQTT server and the Web Interface.

Setup

Once registered you can setup devices. Firstly, generate a unique key via the manage devices section. Modify the source code with your credentials:

  • mqttUser = account username
  • mqttPassword = account password
  • deviceID = unique key
Now compile the source code onto the device and you are ready to go.
